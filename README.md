# Milliarium 📋

A modern Kanban/task management app built with Next.js and Supabase.

## 🌟 Live Website

Visit the application at: [https://milliarium.vercel.app](https://milliarium.vercel.app)

## 🚀 Features

- ✅ User Authentication
- 📊 Kanban Board
- 🎨 Dark/Light Mode
